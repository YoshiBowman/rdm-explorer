Features

Everything you need to know your rig

⬡

Automatic Discovery

Broadcasts ArtPoll to find all Art-Net nodes on your network, then runs the full RDM binary tree discovery to enumerate every connected fixture.

📋

Deep Fixture Info

Reads manufacturer, model, DMX address, personality, footprint, software version, and more — directly from the fixture over RDM.

✏️

Remote Control

Set DMX start addresses, update device labels, and trigger identify mode without touching the fixture — from anywhere on the network.

🔌

Pathport Native

Built with Pathway Pathport nodes in mind. Supports multi-universe scanning across all output ports automatically.

🗂

Filter & Sort

Search across any field, filter by product category, and sort by DMX address, manufacturer, or label to find what you need fast.

⚡

Demo Mode

No hardware? No problem. Load a full rig of realistic demo fixtures to explore the interface before you're on site.

How it works

From launch to full rig in seconds

1

Select your network interface

Choose the Ethernet interface connected to your Art-Net network from the dropdown. On most setups this is automatic.

2

Scan the network

Click Scan Network. RDM Explorer broadcasts ArtPoll to find your Pathport nodes, then sends RDM discovery through every output universe on each node.

3

Fixtures appear as they're found

Devices populate in real time as discovery runs. Each card shows manufacturer, model, DMX address, personality, and footprint.

4

Click any fixture to inspect and control it

Open a fixture's detail panel to see everything RDM has to offer, set a new DMX address, rename the device, or trigger identify.

Install

Pick your platform

🍎 macOS

  1. Download the .dmg file
  2. Open it and drag RDM Explorer to Applications
  3. Launch from Applications

🖥 Windows

  1. Download the .exe installer
  2. Run the installer
  3. Launch RDM Explorer from the Start menu

🐧 Linux

  1. Download the .AppImage
  2. Make it executable:
    chmod +x RDM*.AppImage
  3. Run it: ./RDM*.AppImage
